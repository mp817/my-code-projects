## 知乎文章同步扩展

这是一个Chrome扩展，可以将知乎上的文章同步到其他平台，如CSDN、博客园、掘金和头条。

### 功能特点

- 一键同步知乎文章到多个平台
- 支持提取文章标题、内容、作者、标签等信息
- 用户友好的界面
- 可配置多平台账号信息
