# Flappy Bird 游戏

这是一个使用HTML, CSS和JavaScript实现的Flappy Bird游戏克隆版。

## 游戏特点

- 经典的Flappy Bird游戏玩法
- 响应式设计，适配不同屏幕尺寸
- 本地存储最高分记录
- 简洁美观的界面

## 如何游玩

1. 点击屏幕或按空格键使小鸟飞翔
2. 避开绿色管道
3. 尽可能获得高分

## 技术栈

- HTML5 Canvas用于游戏渲染
- 纯JavaScript实现游戏逻辑
- CSS3实现界面样式

## 本地运行

只需将项目文件下载到本地，然后在浏览器中打开`index.html`文件即可开始游戏。

## 预览

游戏包含开始界面、游戏界面和结束界面，玩家可以随时重新开始游戏并查看自己的最高分。
